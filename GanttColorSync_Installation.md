# התקנת סנכרון גאנט — מדריך מהיר
# Gantt Color Sync — Installation Guide

---

## ⚠️ דרישות מקדימות
- יש להיות **בעל הגיליון** (owner) או בעל **הרשאת עריכה** (editor)
- הקובץ צריך להיפתח בחשבון **office1.aido@gmail.com** (הבעלים)
- אם הגיליון שמור כ-`.xlsx` — מומלץ לשמור קודם כגיליון Google Sheets (ראה סעיף 0)

---

## שלב 0 — (מומלץ) המרה לגיליון Google Sheets

> בצע רק פעם אחת. מבטל את תווית ה-XLSX ומאפשר כל יכולות.

1. פתח את הגיליון בחשבון הבעלים
2. `קובץ` → `שמור כגיליון אלקטרוני של Google`
3. גיליון חדש ייפתח בכרטיסייה — המשך ממנו

---

## שלב 1 — פתח את עורך Apps Script

```
תפריט: הרחבות  →  Apps Script
```

(אם אין "הרחבות" — נסה `כלים` → `עורך סקריפטים`)

---

## שלב 2 — הדבק את הקוד

1. מחק את כל הטקסט הקיים ב-`Code.gs`
2. פתח את הקובץ **`GanttColorSync.gs`** שבתיקייה זו
3. **העתק את כל התוכן** (Ctrl+A, Ctrl+C)
4. חזור לעורך Apps Script והדבק (Ctrl+V)
5. לחץ **שמור** (Ctrl+S)

---

## שלב 3 — הרץ את `onOpen` לראשונה (הענקת הרשאות)

1. בתיבת הבחירה בראש העמוד בחר את הפונקציה `onOpen`
2. לחץ **▶ הפעל**
3. בחלון ההרשאות שייפתח — אשר (Allow / אפשר)
4. חזור לגיליון — תראה תפריט חדש: **🔄 סנכרון גאנט**

---

## שלב 4 — התקן את הטריגר האוטומטי

בגיליון:
```
🔄 סנכרון גאנט  →  התקן טריגר אוטומטי
```

אשר את חלון ההרשאות (אם יופיע שוב).

**זהו.** מהרגע הזה:
- כל צביעת תא בעמודות הגאנט ← מעדכנת אוטומטית D, E, F
- אין צורך ללחוץ כלום

---

## שימוש יומיומי

| פעולה | תוצאה |
|-------|--------|
| צבע תאים בגאנט | D / E / F מתעדכנים אוטומטית |
| **🔄 סנכרון גאנט → סנכרן עכשיו** | עדכון ידני (גיבוי) |
| **🔄 סנכרון גאנט → אמת מבנה שבועות** | בדיקת תקינות עמודות א–ה |

---

## לוגיקה — מה הסקריפט עושה

```
לכל שורת משימה (שורות 9 ומטה, לא כותרות):
  סרוק את כל תאי הגאנט (עמודה I ואילך)
  מצא את התא הצבוע הראשון → D (תאריך התחלה)
  מצא את התא הצבוע האחרון → E (תאריך סיום)
  ספור את כל התאים הצבועים  → F (משך — ימי עבודה)
```

**"צבוע"** = כל צבע שאינו לבן / אפור-בהיר / ריק

תאריך = נלקח משורה 5 (אחד לכל עמודה)

---

## אם משהו לא עובד

| בעיה | פתרון |
|------|--------|
| תפריט 🔄 לא מופיע | הרץ `onOpen` ידנית מהעורך |
| D/E לא מתעדכנים | בדוק שהטריגר מותקן: עורך Apps Script → שעון ← Triggers |
| F לא מתעדכן | וודא שהשורה לא "כותרת סעיף" (מספר שלם כמו 1, 2, 3) |
| שגיאת הרשאות | הרץ שוב `onOpen` ואשר הרשאות |
| תאים צבועים לא מזוהים | הוסף את צבע הרקע לרשימת `EMPTY_BG` ב-`CFG` |

---

## כוונון מתקדם (CFG)

אם פריסת הגיליון משתנה, ערוך את הקטע `const CFG = { ... }` בראש הקובץ:

```javascript
SHEET_NAME:     'גאנט עבודה FLL שנתי',  // שם הגיליון
DATE_ROW:       5,   // שורה עם תאריכי העמודות
FIRST_DATA_ROW: 9,   // שורת המשימה הראשונה
FIRST_COL:      9,   // עמודה I  — תחילת הגאנט
COL_START:      4,   // עמודה D  — תאריך התחלה
COL_END:        5,   // עמודה E  — תאריך סיום
COL_DURATION:   6,   // עמודה F  — משך
```
